== README

PRINiT is an e-commerce platform on which users can upload images, and users purchase canvas prints of their own images or those of other users'.


== Implementation

* AngularJS

* Ruby on Rails

* Rspec testing

* MongoDB

* Stripe API

* Printful API 

* SASS 

* javascript

* Bootstrap

* jQuery
